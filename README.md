# Danrex Vault

Менеджер паролей от **Danrex Corp** — софт-направления **Danrex Tools**.

---

## Возможности

- Хранение паролей в зашифрованной базе данных (AES-256)
- Генератор надёжных паролей
- Импорт и экспорт CSV
- Мастер-пароль при запуске
- Копирование пароля в буфер обмена
- Категории и папки для паролей (премиум)
- Защищённые заметки (премиум)
- Поиск по названию и логину

## Системные требования

- Windows 7 / 8 / 10 / 11
- Установка не требуется

---

## Как использовать

1. Скачайте **Danrex Vault.exe** из раздела Releases
2. Запустите
3. Придумайте мастер-пароль
4. Добавляйте пароли через кнопку «+»

---

## Безопасность

- Все пароли шифруются алгоритмом AES-256-CBC
- Мастер-пароль хешируется через SHA-256 с солью
- Пароли хранятся только локально на устройстве
- Приложение не отправляет данные в интернет
